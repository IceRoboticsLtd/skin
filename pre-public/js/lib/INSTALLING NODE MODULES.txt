To install node modules, run the following command whilst in the directory where package.json resides:

npm install

To update node modules, run the following command whilst in the directory where package.json resides:

npm update

NOTE: bower is a node module. It is its own package manager for browser-based modules, unlike npm which is a package manager for server-based modules.


TIP: To clear npm's cache when running out of space or wanting to guarantee that all required modules are properly installed, run the following command:

npm cache clear
